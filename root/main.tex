**Theorem.** The set of natural numbers equal to the sum of the factorials of their own decimal digits is exactly $\{1, 2, 145, 40585\}$. Formally, if we define $f(n) = \sum_{d \,\in\, \text{digits}(n)} d!$, where $\text{digits}(n)$ denotes the list of decimal digits of $n$, then

$$\{\, n \in \mathbb{N} \mid f(n) = n \,\} = \{1,\, 2,\, 145,\, 40585\}.$$
